# Si1eX

This is a Next.js application that showcases a developer's portfolio. It features a navigation bar, and a brief introduction to the developer.

## Getting Started

To get started, clone the repository and install the dependencies:

```
git clone https://github.com/si1ex/si1ex.git
cd si1ex
npm install
```

Then, start the development server:

```
npm run dev
```

The application will be available at http://localhost:3000.

## Code Explanation

The code for this application is located in the `src/app` directory. The main component is the `RootLayout` component, which is responsible for rendering the layout of the application. The `RootLayout` component imports the `Nav` component, which renders the navigation bar, and the `Particles` component, which renders the particle background.

The `Home` component is the main page of the application. It renders a brief introduction to the developer and a link to their portfolio.

## Conclusion

This application is a simple example of how to create a developer portfolio using Next.js. It demonstrates how to use Next.js to create a custom layout, add a粒子背景, and render a navigation bar.
